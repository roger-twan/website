## ‚öíÔ∏è Development

Create a `.env.local` file on the root directory and add `GITHUB_TOKEN` to it.

```bash
# install dependencies
npm install

# run server
npm run dev

# run unit test
npm run test
```

Open [http://localhost:3000](http://localhost:3000) in the browser to view the result.
